Monochrome Particle

A small skills pack for coding agents: implement a full-viewport Three.js layered particle field with explicit controls for gradient colors, animation speed, flow direction, density, point size, and opacity. Scope is the ambient canvas only — not page transitions or CTA wiring. The content is plain Markdown and TypeScript; use it from Cursor, Claude Code, Codex, Gemini, Windsurf, JetBrains AI, Copilot, VS Code–based agents, or any workflow where you attach or paste the skill into the model context.

Install

Use the Agent Skills CLI with GitHub shorthand (shortest):

npx skills add itsyuvallavi/monochrome-particle

Same as npx skills add https://github.com/itsyuvallavi/monochrome-particle — the CLI accepts owner/repo.

Optional filter (if your CLI supports --skill): npx skills add itsyuvallavi/monochrome-particle --skill monochrome-particle

LLM-friendly knobs

Users should be able to ask in plain language, for example:

  • “Slow the motion down” → speed
  • “Drift up and left” → direction
  • “More particles” → density
  • “Cooler palette” → colors.start|mid|end

See examples/MonochromeDotsBackground.tsx in the repo.
