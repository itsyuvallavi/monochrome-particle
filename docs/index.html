Monochrome Particle

A portable skill for coding agents: build a full-viewport Three.js layered particle wave with clear knobs for colors, speed, direction, density, point size, and opacity. Background only — no route transitions or CTA wiring.

Install (CLI)

After you publish the GitHub repository, replace the URL with your repo. 

npx skills add https://github.com/YOUR_USERNAME/monochrome-particle

What you get

  • skills/monochrome-particle/SKILL.md — agent instructions
  • skills/monochrome-particle/reference.md — shader and performance reference
  • skills/monochrome-particle/examples/MonochromeDotsBackground.tsx — copy-paste React example
